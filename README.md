# NextGen Crypto Hub

A modern, blazing-fast multi-chain cryptocurrency portfolio tracker, trading dashboard, and DeFi insights platform.

Built with **Next.js 15**, **wagmi/viem**, **RainbowKit**, and AI-powered analytics.

## ✨ Features

- Multi-chain support (EVM + Solana)
- Real-time portfolio valuation
- AI-driven market insights & risk analysis
- DeFi yield aggregator
- Beautiful charts & dark mode
- Fully non-custodial (you control your keys)
- PWA-ready for mobile

## 🚀 Quick Start

```bash
git clone https://github.com/YOURUSERNAME/nextgen-crypto-app.git
cd nextgen-crypto-app
npm install
cp .env.example .env.local
# Add your CoinGecko API key, RPC URLs, etc.
npm run dev
